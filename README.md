# Bank System Project (C#)

Консольное приложение для управления банковскими счетами с модульной структурой.

## Функционал:
- **Saving Account**: Расчет месячной прибыли и контроль минимального остатка.
- **Premium Account**: Поддержка овердрафта и фиксированная комиссия.
- **Investment Account**: Управление виртуальными активами (Stocks, Bonds, ETF) и прогноз годового роста.

## Как запустить:
1. Установите .NET SDK.
2. Скомпилируйте проект: `dotnet build`.
3. Запустите: `dotnet run`.
