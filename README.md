# Bank System (Golang Edition) 🚀

Простая консольная система управления банковскими счетами, полностью переписанная с **C#** на **Go**. Этот проект демонстрирует переход от объектно-ориентированного программирования (ООП) к структурному подходу и композиции в Golang.

## 📁 Структура проекта

Код разделен на логические модули для удобства навигации и чистоты архитектуры:

* **`models.go`** — описание структур данных `Client` и `Bank` (аналоги классов в C#). Содержит поля для хранения имен, паролей и списков счетов.
* **`bank.go`** — ядро системы: логика регистрации клиентов, аутентификация (вход в систему) и генерация уникальных номеров счетов вида `ACC-XXX`.
* **`main.go`** — точка входа в приложение. Содержит интерактивное консольное меню и главный цикл обработки команд пользователя.
* **`go.mod`** — файл инициализации модуля, объединяющий все файлы в единый исполняемый проект.

## 🛠 Функционал

Приложение имитирует базовые операции банковского сервиса:
1.  **Регистрация**: Создание профиля клиента с привязкой контактных данных.
2.  **Управление счетами**: Автоматическое создание счетов с отслеживанием их количества для каждого клиента.
3.  **Безопасность**: Проверка пароля при входе в систему (аутентификация).
4.  **Финмониторинг**: Автоматическая проверка транзакций на подозрительно высокие суммы (свыше 100,000).

## 🚀 Как запустить у себя

1.  Убедитесь, что у вас установлен [Go](https://go.dev/dl/) версии 1.18 или выше.
2.  Склонируйте репозиторий:
    ```bash
    git clone [https://github.com/pokimon2178-star/Bank-System-with-modular-structure.git](https://github.com/pokimon2178-star/Bank-System-with-modular-structure.git)
    ```
3.  Перейдите в папку проекта:
    ```bash
    cd Bank-System-with-modular-structure
    ```
4.  Запустите приложение:
    ```bash
    go run .
    ```

## 💡 Чему я научился при переходе с C# на Go
* **Композиция вместо наследования**: Использование структур (`struct`) и методов вместо иерархии классов из C#.
* **Управление данными**: Работа с картами (`map`) для быстрого поиска объектов вместо `Dictionary<T>`.
* **Модульность**: Организация кода в рамках одного пакета (`package main`) для бесшовной связи нескольких файлов.
